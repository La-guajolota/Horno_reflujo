## Descripción General

Este proyecto consiste en el diseño y construcción de un **horno de reflujo** para soldadura de componentes SMD en placas PCB. El horno permite controlar la temperatura de manera precisa, siguiendo una curva de reflujo específica para garantizar la correcta soldadura de los componentes.

---

## Características Principales

- **Control de Temperatura Preciso**: Permite seguir una curva de temperatura específica para diferentes tipos de soldadura.
- **Interfaz de Usuario Amigable**: Facilita la configuración y monitoreo del proceso de soldadura.
- **Sistema de Extracción de Gases**: Filtra y extrae los vapores tóxicos generados durante el proceso de soldadura.
- **Diseño Modular**: Facilita el mantenimiento y la actualización de los componentes.
